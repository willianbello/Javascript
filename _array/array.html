• Criando Array de String e de Numeros
  • Utilizando índices e imprimindo valor das variáveis

    • Notas:

    Para declarar um array é necessário utilizar a palavra reservada Array() ou simplesmente []. Podemos declarar os valores na criação como, por exemplo: var numeros = Array (1, 5, 10); ou var numeros = [1, 5, 10];. Podemos declarar os valores depois da criação como, por exemplo: var numeros = Array (); ou var numeros = []; e depois:

    • numeros[0] = 1;
    • numeros[1] = 5;
    • numeros[2] = 10;
